How it works

Each plugin reads its demo-library.json manifest from its folder above. The manifest lists demos with preview images and data; importing a demo downloads its content and images into the site's Media Library.

This repository hosts only demo resources for my WordPress plugins. No plugin source code is stored here, and these files are not bundled into any WordPress.org plugin package.
