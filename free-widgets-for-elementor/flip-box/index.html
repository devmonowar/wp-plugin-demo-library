Features

Flip Box Row

Three interactive flip boxes that reveal a back face with a call to action.

How to import

  1. Install and activate Free Widgets For Elementor (and Elementor) on your site.
  2. Go to Free Widgets → Demo Library in wp-admin.
  3. Find Flip Box Row and click Import — the section is added to Templates → Saved Templates and its images to your Media Library.
  4. Edit any page with Elementor and insert the saved template wherever you like.
